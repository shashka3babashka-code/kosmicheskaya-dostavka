# kosmicheskaya-dostavka
## Описание проекта
Учебный MVP мобильной 2D-игры. Игрок управляет марсианским курьерским ровером и выполняет доставки между поселениями.

## Цель игрока
За 3–5 минут выбрать заказ, проверить энергию, доставить посылку и получить очки репутации.

## Жанр
Казуальный симулятор / менеджер ресурсов.

## Основные механики
- Запуск миссии
- Карта сектора
- Выбор заказа
- Проверка энергии
- Доставка
- Начисление очков
- Зарядка ровера
- Экран результата

## Сцены
- Стартовое меню
- Карта сектора
- Экран результата
- Экран паузы

## Ассеты
Папка `assets/` (изображения, звуки, шрифты).
